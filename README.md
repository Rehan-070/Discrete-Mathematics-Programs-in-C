# 🧮 Discrete Mathematics Programs in C

This folder contains C programs implementing basic concepts of **Discrete Mathematics** such as set operations, logical operations, and other fundamental topics.  
The programs are designed for learning and practicing how mathematical concepts can be implemented using the C programming language.

---

## 🚀 Topics Covered

- 🔢 Set Operations
  - Union
  - Intersection
  - Difference
  - Subset checking

- 🧠 Logical Operations
  - AND
  - OR
  - XOR
  - NOT

- 🔀 Bitwise Operations
  - Left Shift
  - Right Shift

- 📊 Basic Discrete Mathematics Concepts
  - Boolean logic
  - Truth-style evaluations
  - Mathematical computation examples

---

## 🗂️ Folder Structure

Example files in this folder:

set_operations.c  
logical_oprators.c  
bitwise_operations.c  

Each program demonstrates a specific concept of **Discrete Mathematics using C**.

---

## 🛠️ Requirements

Make sure you have a **C compiler** installed.

Check GCC version:

`gcc --version`

For Windows:
- Install **MinGW** or use **CodeBlocks**

For Linux/Mac:
- GCC is usually pre-installed.

---

## ▶️ How to Run a Program

# compile the file  
`gcc filename.c -o output`

# run the program (Windows)  
`output`

# run the program (Linux/Mac)  
`./output`

Example:

`gcc logical_oprators.c -o logical_oprators`  
`./logical_oprators`
